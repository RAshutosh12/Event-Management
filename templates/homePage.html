Event Management Portel

Ola, Welcome to Event Management Portel!!!

This is a portel where you can create events and there sub events.

Features of this portel:-

- Create an Event with it's dates.

- Create sub Events of Parent Event.

- Manage dates and add descriptions of each event and other informations.

- you will get a notification mail of event that need to start on set date or you can adjust that also.
